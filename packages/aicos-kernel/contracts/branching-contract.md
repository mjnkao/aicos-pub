# Branching Contract

Status: initial MVP contract

Project branches live under:

```text
brain/projects/<project-id>/branches/<branch-id>/
```

Each branch should contain at least:

- `branch-profile.md`
- `inherited-context.md`
- `assumptions.md`
- `recommendation.md`

Branches are experiment reality. They do not override project canonical or
working reality unless reviewed and promoted.
