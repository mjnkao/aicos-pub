# Rule Card: Idea Capture

Load when new ideas, TODOs, tech debt, or branch ideas appear.

## Route

- open question: `brain/projects/aicos/working/open-questions.md`
- clear later task: `agent-repo/classes/a2-service-agents/tasks/backlog/`
- candidate branch idea: `brain/projects/aicos/evidence/candidate-decisions/`
- small tech debt: A2 backlog
- tech debt with real impact: `brain/projects/aicos/working/active-risks.md`
- committed decision: working, approval, or branch lane

## Rule

Do not lose useful ideas, but do not put every idea into `current-state.md`.

Do not create official branch reality until a human/manager choice or approved
AICOS flow selects that branch to try.
