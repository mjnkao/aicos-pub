# A2 Idea Capture Rules

Status: reference/deep policy, not startup-default

Do not lose useful ideas, but do not turn `working/current-state.md` into an
inbox.

## Routing

- Open question: `brain/projects/aicos/working/open-questions.md`
- Clear later task: `agent-repo/classes/a2-service-agents/tasks/backlog/`
- Candidate branch idea not yet chosen:
  `brain/projects/aicos/evidence/candidate-decisions/`
- Small tech debt: A2 backlog
- Tech debt with architecture or operational impact:
  `brain/projects/aicos/working/active-risks.md`
- Committed decision: the relevant working, approval, or branch lane

## Branch Ideas

Do not create official branch reality for an unapproved branch idea. Create
`brain/projects/aicos/branches/<branch-id>/` only after a human/manager choice
or an approved AICOS flow selects that branch to try.
