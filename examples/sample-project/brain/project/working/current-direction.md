# Sample Research Digest Current Direction

Status: public sample

## Direction

Use this project to teach the AICOS operating model:

1. Start from project identity and authority boundaries.
2. Read role-appropriate context.
3. Select one task packet.
4. Inspect source artifacts only when needed.
5. Write meaningful continuity back to AICOS.

## Next Step

Create a sample task packet for reviewing the default digest slice.

## Non-Goals

- Do not turn the sample into a full product.
- Do not add private data.
- Do not import large generated artifacts.
