# Tóm Tắt Service Skills Đang Làm Việc

Trạng thái: working summary.

A2 service skills hiện là markdown contracts trong:

```text
agent-repo/classes/a2-service-agents/skills/
```

Đã có:

- `resolve-scope`
- `build-project-capsule`
- `build-branch-capsule`
- `generate-mvp-options`
- `compare-branches`
- `recommend-promotion`
- `improve-query-patterns`

Hiện tại các skill này là policy/service scaffolds, chưa phải runtime tự động.
Điều này phù hợp vì A2-Serve chưa active đầy đủ và service intelligence còn
đang thay đổi.
