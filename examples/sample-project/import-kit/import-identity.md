## AICOS Identity

- AICOS project id: `sample-project`
- AICOS authority: context/control-plane, compact digests, working state, task

## Context Scope Mode

Active context means current project truth, current workstream, current state,
selected task packets, and role-aware startup surfaces.

Reference/history means raw source notes, old generated outputs, previous
handoffs, logs, and broad archives.
