# Sample Research Digest Source And Data Scope

Status: public sample

## Source Roles

- Public notes: primary source material.
- Public links: supporting references.
- Human annotations: reviewed context and priority signals.
- Generated digest outputs: artifacts, not source truth.

## Runtime Package Policy

Digest packages should be compact, source-attributed, role-aware, and explicit
about missing or stale sources.

They should not become:

- raw source mirrors;
- private memory dumps;
- hidden ranking engines;
- fabricated certainty.

## Data Semantics

- Raw facts: note title, source path, timestamp, link, excerpt summary.
- Derived facts: topic tags, freshness status, coverage gaps.
- Structured descriptors: priority, confidence, unresolved question, next read.
- Final synthesis remains assistant/human reasoning, not hidden pipeline output.
