# Rule Card: A1 Task Packet Loading

Load when choosing or starting a concrete A1 task.

## Rule

- Start from orientation first.
- Load one full task packet only after the task is selected or strongly implied.
- Load only the rule cards named by the packet.
- If no task packet exists, create or request the smallest task state needed
  before doing broad work.
- Treat project, workstream, work context, primary artifact(s), and task as
  separate fields.

Do not load all packets, all evidence, or all handoffs at startup.
