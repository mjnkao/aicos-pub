# Sample Project Profile

Status: template
Project id: `sample-project`
Scope: `projects/sample-project`

## Purpose

`sample-project` is a synthetic AICOS project brain used to demonstrate the
minimum context/control-plane shape for a real project.

## Authority Boundary

- This file describes project identity and durable scope.
- Working files describe current mutable state.
- Evidence files preserve source inputs and research.
- Runtime systems, issue trackers, repos, and dashboards remain outside AICOS
  unless referenced through artifact refs.

## Actors

- Human manager: assigns priorities and accepts decisions.
- A1 external agent: performs project work and uses AICOS for context and
  continuity.
- A2 maintainer: updates AICOS service/template behavior when needed.

## Non-Goals

- This sample does not define a real product.
- This sample does not contain private company, customer, or project data.
