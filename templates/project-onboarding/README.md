# Project Onboarding Kit

Status: shared reusable foundation

This kit is for bringing an existing real project into AICOS without inventing a
one-off import process.

Use these files in order:

1. `import-checklist-template.md`
2. `lane-mapping-template.md`
3. `bootstrap-state-template.md`
4. `startup-token-test-template.md`
5. `rule-compliance-test-template.md`
6. `continuity-interruption-test-template.md`
7. `real-task-quality-test-template.md`
8. `evaluation-scorecard-template.md`

Project-specific answers belong under `brain/projects/<project-id>/`, not in
these shared templates.
