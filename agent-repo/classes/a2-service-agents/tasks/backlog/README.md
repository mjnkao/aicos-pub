# A2 Backlog

This lane holds clear A2 tasks, TODOs, and tech debt that should not interrupt
the current flow.

Use this lane when an item is actionable but not the current priority.

When a backlog item becomes current, blocked, or waiting, use the lightweight
task state convention in `agent-repo/classes/a2-service-agents/tasks/README.md`
so another actor can continue it later without reading the whole session.

Do not use this lane for:

- canonical truth
- active project working state
- unreviewed branch reality
- raw chat transcript
