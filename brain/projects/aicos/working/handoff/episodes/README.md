# Episodic Handoffs Của AICOS

Lane này dành cho H2 episodic handoff notes: các pass/test/refinement có phạm
vi rõ và cần giữ provenance cho agent sau.

Quy ước tên file:

```text
YYYY-MM-DD_<actor-lane>_<topic>.md
```

Ví dụ:

```text
2026-04-18_a2-core_storage-structure-normalization.md
```

Không đọc toàn bộ folder này ở startup. Chỉ đọc khi H1 current handoff trỏ tới
hoặc task cần provenance cụ thể.

Các handoff/provenance notes cũ đã được chuyển vào
`backup/handoff-provenance-20260418/`. Chỉ mở backup khi cần audit lịch sử, không
dùng làm startup path.
