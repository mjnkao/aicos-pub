# Hồ Sơ Dự Án AICOS

Trạng thái: canonical tối thiểu, đang dùng cho self-brain.

AICOS là lớp trí tuệ cho company/workspace/project, giúp humans và AI agents
cùng làm việc trên một shared project reality.

AICOS phục vụ:

- human manager, reviewer, approver;
- A1 work agents làm project/business work;
- A2 agents cải thiện chính AICOS;
- nhiều projects và nhiều branches/experiments.

AICOS không phải:

- personal brain cho một human duy nhất;
- task board chung chung;
- backend database làm truth;
- UI-first product trong MVP hiện tại.

Mục tiêu MVP hiện tại là làm AICOS có self-brain, working state, CLI mỏng,
folder boundary rõ, và một flow local `blocked -> options -> manager choice`
có thể review được.
