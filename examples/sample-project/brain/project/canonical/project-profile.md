# Sample Research Digest Project Profile

Status: public sample
Project id: `sample-project`

## Identity

Sample Research Digest is a synthetic project used to demonstrate AICOS project
context management.

The project collects public research notes, normalizes them, and packages a
daily assistant-ready digest.

## Authority Model

- AICOS is the context/control-plane authority for compact project context,
  working state, role-aware onboarding, task packets, risks, open questions, and
  continuity.
- The sample project checkout is the artifact/runtime authority for source
  collectors, scripts, tests, schemas, generated digest outputs, and source
  provenance.
- Raw source notes and generated outputs are evidence until reviewed and
  promoted into compact current context.

## Working Boundary

AICOS should not mirror the whole project repo. It should store concise,
reviewed project context that helps agents enter work safely and quickly.

## Public Role Boundary

AICOS-internal labels such as A1 and A2 describe control-plane actor classes.
External project work should also declare project-facing roles such as product
owner, CTO, fullstack dev, reviewer, or operator.
