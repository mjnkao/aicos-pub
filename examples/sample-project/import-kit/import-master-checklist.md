# Sample Project Import Master Checklist

Status: public sample

## Checklist

- [ ] Identify source repo and branch.
- [ ] Define AICOS project id.
- [ ] Confirm source repo remains artifact/runtime authority.
- [ ] Define AICOS context/control-plane authority.
- [ ] Select canonical-active sources.
- [ ] Select working-active sources.
- [ ] Select delivery-active sources.
- [ ] Explicitly exclude archives/history/generated bulk.
- [ ] Create compact canonical digests.
- [ ] Create working state.
- [ ] Create context ladder.
- [ ] Create task packet index.
- [ ] Create first task packet only after real task selection.
- [ ] Validate role-aware onboarding.
